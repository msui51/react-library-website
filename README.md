## Summary

A responsive book store web app built with React. Users can browse books, sort listings, view book details, and manage a shopping cart.

## Features

- Browse a collection of books
- View featured and discounted books
- Sort books by price and rating
- View individual book details
- Add books to cart
- Update item quantity in cart
- Remove items from cart
- View subtotal, tax, and total

## Built With

- React
- JavaScript
- CSS
- HTML
